# Legal Skills

面向律师办案的 Claude Skills 集合，支持诉讼与非诉场景的 AI 协作。

## 项目概述

本项目旨在沉淀并分发可复用的法律领域技能，将 Claude 从通用 AI 助手转变为专业的法律工作协作者。每个技能都是独立、自包含的模块，可以单独使用或组合使用。

## 技能列表

### 官方技能

来自 Claude Code 官方的通用技能：

| 技能                                   | 说明                                         |
| -------------------------------------- | -------------------------------------------- |
| **[skill-creator](skill-creator/)** | 创建新技能的指南和工具集                     |
| **[pdf](pdf/)**                     | PDF 处理工具包：文本提取、表单填写、合并拆分 |

### 自研技能

面向法律场景定制开发的技能：

| 技能                             | 说明                                          |
| -------------------------------- | --------------------------------------------- |
| **[mineru-ocr](mineru-ocr/)** | PDF/图片转 Markdown，支持 OCR、表格和公式识别 |

## 技能结构规范

每个技能遵循统一的结构：

```
skill-name/
├── SKILL.md               # 技能定义文件（必填）
├── DECISIONS.md           # 重要决策记录
├── TASKS.md               # 任务清单
├── CHANGELOG.md           # 变更日志
├── references/            # 参考资料
├── scripts/               # 可执行脚本
└── assets/                # 资源文件
```

### SKILL.md 格式

每个技能必须包含 YAML frontmatter：

```yaml
---
name: skill-name
description: 技能描述，说明何时使用此技能
license: 许可证信息
---
```

## 协作规范

本项目遵循 [AGENTS.md](AGENTS.md) 定义的协作规范：

- **技能导向**：每个技能独立成树，自包含所有资源
- **文档即上下文**：关键决策、任务、变更必须记录在文档中
- **透明变更**：所有修改写入 CHANGELOG.md
- **保留证据**：输出引用可回溯，缺失信息明确标注
