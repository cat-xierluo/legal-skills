# md2html 更新日志

## 2.1.0 (2026-03-06)

### 新增

- **设计系统**
  - 多套配色主题（default, ocean, forest, sunset, violet）
  - 统一的设计 token（spacing, radius, colors）
  - SVG 图标系统（替代 Emoji）
  - 专业线性图标风格

## 2.0.0 (2026-03-05)

### 新增

- **PDF 一键保存按钮**
  - 浮动按钮，固定在页面右下角
  - 点击触发浏览器打印对话框，可保存为 PDF
  - 滚动时自动半透明，避免遮挡内容
  - 打印时自动隐藏

## 1.0.0 (2026-03-05)

### 新增

- **核心功能**
  - Markdown 转 HTML 可视化报告
  - 自动目录导航生成
  - 表格样式增强（斑马纹、高亮）
  - 法条引用识别与格式化
  - 案件信息卡片检测

- **主题系统**
  - legal：法律文书风格（默认）
  - academic：学术报告风格
  - minimal：极简风格

- **打印优化**
  - 打印时隐藏目录
  - 优化页边距
  - 分页控制

- **命令行工具**
  - 单文件转换
  - 批量转换
  - 主题选择
  - 目录生成选项
