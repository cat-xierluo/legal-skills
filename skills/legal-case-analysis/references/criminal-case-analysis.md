# 刑事案件分析法

本文件用于刑事案件的研判与审查，是本 Skill 的刑事分析支架，与民事/民商事的 `element-litigation-nine-step.md` 并列使用。

它以**审查起诉视角为骨架**（承办检察官如何审查一个案件），同时用**辩护对抗检验**横切每一步（如果站在辩护人立场，出罪、罪轻、程序瑕疵的空间在哪里）。这两个视角互为镜像：辩护对抗检验既是律师本职，也是检察官防错案的自我审查方法。

本文件不声称替代某一固定学术版本；如团队已有特定版本，以团队版本为准。具体法条、司法解释、量刑指导意见以 `yuandian-law-search` 检索结果为准，不凭记忆写确定结论。

> **输入完整度决定产出深度**：输入若仅为案情概述/侦查简报/起诉意见书叙述（无完整卷宗证据），产出限于"初步研判 + 四要件框架 + 证据缺口清单 + 补证方向"，**不得下确定定罪结论或确定量刑**；输入为完整卷宗（含言词证据、物证、鉴定、勘验、程序文书）时，方可产出确定定性与量刑建议。这条防止在材料不足时强行给出确定结论。

## 一、适用场景

出现以下情况时使用：

- 刑事案件研判、审查逮捕/审查起诉底稿、辩护意见底稿。
- 定性分析（此罪与彼罪、一罪与数罪、既未遂、共犯地位）。
- 证据审查与证据链评估、非法证据识别。
- 量刑分析与认罪认罚评估。
- 判断是否符合起诉条件、是否需要补充侦查。

**不适用/请转其他支架**：

- 民事、民商事请求权争议 → `element-litigation-nine-step.md`。
- 起诉书、辩护词、量刑建议书等成稿诉讼文书 → 相应文书生成 Skill（如 `legal-proposal-generator`）。

## 二、犯罪构成框架说明

- 本支架**默认采用四要件**（犯罪客体、犯罪客观方面、犯罪主体、犯罪主观方面）——这是中国司法实务（起诉书、判决书定罪说理）的通行口径，检察官、法官、律师日常使用。
- **违法/责任阻却事由用三阶层（构成要件该当性 → 违法性 → 有责性）组织审查**：正当防卫、紧急避险等属"违法性"层，责任能力、违法性认识可能性、期待可能性等属"有责性"层；这是出罪、罪轻和防错案的必经实务审查，在第三步清单处逐项过一遍。
- 同一份分析内框架不得反复横跳；默认四要件做定罪主线，阻却事由按三阶层归位评价并标注。

## 三、刑事分析八步法

### 第一步：明确角色、目标与程序阶段

- **角色**：承办检察官（审查起诉骨架）；同时标注辩护对抗检验视角。若实际委托为辩护/代理，则以辩护为主、审查视角作反向检验。
- **目标**：是否构成犯罪、定性、是否起诉、量刑、程序处理、是否补充侦查。
- **程序阶段**：侦查/审查逮捕/审查起诉/一审/二审/再审——阶段决定审查重点与证据标准。

### 第二步：事实梳理与时间线

**先甄别材料性质**：判断输入是 ①完整卷宗、②案情概述/侦查简报、③起诉意见书/起诉书的事实叙述、④混合。②③本身**不是法定证据**，只是办案机关的单方叙述——只能作为"待核实的事实线索"，不得作为定案依据；据此产出的只能是"初步研判 + 证据清单"，不能下确定定罪结论。

将材料转化为事实，而非直接摘要。每条事实至少含：时间、主体、行为或事件、来源材料、证据强度、是否存在争议。

区分**证据事实 / 犯罪嫌疑人供述 / 证人证言 / 存疑事项**；单方供述不得直接写成定案事实（呼应本 Skill 核心原则）。对未提及但影响判断的信息，标注"未提及/待补充"。

### 第三步：定罪分析（四要件逐项对照）

逐要件把事实与证据映射进来：

| 要件 | 审查内容 | 对应事实 | 支撑证据 | 是否满足 |
|------|----------|----------|----------|----------|
| 犯罪客体 | 侵害的法益 | | | 是/否/待核实 |
| 客观方面 | 危害行为、危害结果、因果关系、特定时间地点方法等情节 | | | |
| 犯罪主体 | 刑事责任年龄、责任能力、特殊身份、单位犯罪 | | | |
| 主观方面 | 故意/过失、犯罪目的、明知、认识内容 | | | |

涉及违法/责任阻却事由或疑难定性时，按下方三阶层（违法性→有责性）逐层审查，结论回填本表。

**违法/责任阻却事由核查清单**（四要件初步满足后，按三阶层逐层过一遍，命中则在相应位置评价）：
- 违法性层（违法阻却）：正当防卫、防卫过当、紧急避险、被害人承诺、法令行为/正当业务行为。
- 有责性层（责任阻却）：无/限制刑事责任能力（年龄、精神状态）、违法性认识可能性、期待可能性。

命中任一事由，即在该阶层位置展开评价、说明出罪或减轻责任的理由，结论回填上表。

### 第四步：罪数与犯罪形态

- **此罪与彼罪**：易混淆罪名的界分（如故意伤害 vs 寻衅滋事、诈骗 vs 民事欺诈）。
- **一罪与数罪**：想象竞合、牵连犯、吸收犯、连续犯、数罪并罚。
- **犯罪形态**：预备、未遂、中止、既遂。
- **共同犯罪**：主犯/从犯/胁从犯、教唆犯、帮助犯，各参与人地位与作用。

### 第五步：证据审查（刑事证明标准）

刑事证明标准是**事实清楚、证据确实充分、排除合理怀疑**——远高于民事的高度盖然性。逐项审查：

- **法定证据种类核查**（《刑事诉讼法》第 50 条八类）：物证 / 书证 / 证人证言 / 被害人陈述 / 犯罪嫌疑人供述和辩解 / 鉴定意见 / 勘验检查辨认侦查实验等笔录 / 视听资料电子数据。逐类核对哪些已到案、哪些缺失；非法定种类的材料（如"案情概述"）不得单独作为定案依据。
- **三性**：真实性、合法性、关联性（与本 Skill 通用规则一致）。
- **证据链闭合与印证**：各证据能否相互印证、形成完整闭合链条。
- **口供补强**：不轻信、不偏信口供；只有口供没有其他证据不得定案。
- **非法证据排除**：刑讯逼供、威胁、引诱、欺骗取得的供述，违反法定程序取得的物证书证。
- **矛盾检测**：多次供述之间、供述与证人证言之间、言词证据与客观证据之间的矛盾。

按定罪要件建立证据-缺口表：

| 定罪要件 | 证明对象 | 现有证据 | 证明力 | 是否达到"确实充分" | 缺口/补证方向 |
|----------|----------|----------|--------|--------------------|----------------|
| 要件 | 事实 | 证据 | 强/中/弱 | 是/否 | |

### 第六步：量刑分析

- **法定刑幅度** → 量刑起点 → 基准刑 → 情节调节 → 宣告刑区间。
- **法定情节**：自首、坦白、立功、累犯、未成年人、限制责任能力、犯罪形态（未遂/中止等）、主从犯。
- **酌定情节**：退赃退赔、赔偿谅解、认罪态度、前科劣迹。
- **认罪认罚从宽**：是否适用、从宽幅度、量刑建议（确定刑/幅度刑）。
- **缓刑**：是否符合适用条件。

量刑区间与情节以检索到的量刑指导意见为准。

### 第七步：程序与强制措施合法性

- **管辖**与**办案期限**。
- **强制措施**：逮捕必要性/社会危险性评估；强制措施是否适当、期限是否合法。
- **诉讼权利保障**：辩护权、值班律师、认罪认罚具结。
- **侦查活动合法性**：是否存在程序违法、需要监督纠正之处。
- **涉案财物与违法所得**：查封扣押冻结是否合法、处置意见。

### 第八步：结论与处理意见

- **定性结论**与成立可能性（有利/不利因素并列）。
- **证据缺口与补证/退回补充侦查方向**。
- **处理意见**（按条件判断，不只罗列）：
  - **起诉**：犯罪事实清楚、证据确实充分、依法应追究刑事责任。
  - **法定不起诉**（《刑诉法》第 16 条）：情形之一成立（如不构成犯罪、情节显著轻微危害不大、已过追诉时效等）。
  - **相对不起诉**（第 177 条第 2 款）：犯罪情节轻微、依刑法不需要判处刑罚或免除刑罚。
  - **存疑不起诉**（第 175 条第 4 款）：二次退回补充侦查后证据仍不足、不符合起诉条件（疑罪从无）。
  - **附条件不起诉**（第 282 条）：未成年人、可能判一年有期徒刑以下刑罚、有悔罪表现且符合起诉条件。
  - **退回补充侦查**（第 175 条）：证据不足或事实不清，可退补（至多二次）。
- **量刑建议**。
- **辩护对抗检验小结**：辩方最可能攻击的环节——定性往轻、证据链薄弱点、非法证据、程序瑕疵、法定从轻情节。
- **风险等级**（高/中/低 + 理由）。

## 四、辩护对抗检验（横切每一步）

每一步都做一次"辩方视角"反问，找出罪、罪轻、程序救济空间：

- 是否存在违法阻却/责任阻却事由（正当防卫、紧急避险、责任能力等）。
- 定性是否可往轻罪主张（此罪 vs 彼罪）。
- 证据是否达不到"确实充分、排除合理怀疑"。
- 是否有非法证据可申请排除。
- 是否有被遗漏的法定/酌定从轻情节。

## 五、最小输出格式

刑事场景至少保留以下表格 + 结论：

| 罪名 | 四要件对照 | 证据是否确实充分 | 主要缺口 | 罪数/形态/共犯 | 量刑区间 | 处理意见 | 风险 |
|------|-----------|------------------|----------|----------------|----------|----------|------|
| 罪名 | 要件满足情况 | 是/否 | 缺口 | | | 起诉/不诉/补侦 | 高/中/低 |

案件复杂时，将该表作为完整法律分析报告或审查报告底稿的前置底稿。

## 六、边界

- 不替代法律检索：涉及具体罪名构成、司法解释、量刑指导意见时，先用 `yuandian-law-search` 检索再分析。
- 成稿诉讼文书（起诉书/辩护词/量刑建议书等）→ 相应文书生成 Skill。
- 材料未提及的信息标注"未提及/待补充"，不自行补全为定案事实。
