# Changelog

All notable changes to this skill will be documented in this file.

## [1.0.2] - 2026-02-15

### 新增
- 上下文智能匹配机制：基于技术领域、功能描述、技术栈进行仓库识别
- 支持处理同名或名称相似的项目
- 新增"需人工确认"报告类别，当存在多个候选时展示推荐列表

### 改进
- 完善工作流程文档，增加上下文分析步骤
- 输出报告中包含匹配依据说明

## [1.0.1] - 2026-02-15

### 新增
- 添加 `config/.env.example` 环境变量配置模板
- 在 SKILL.md 中添加依赖章节，说明系统依赖和环境变量配置

## [1.0.0] - 2026-02-15

### 新增
- 初始版本发布
- 支持从文章、截图、文字中提取 GitHub 仓库并自动 star
- 支持多种内容来源：URL、文字内容、图片截图
- 支持 GitHub 搜索功能按名称查找仓库
- 自动检查仓库是否已 star，避免重复操作
- 生成详细的 star 报告

### 技术优化
- 使用 gh CLI 进行 GitHub 认证
- 遵循 AGENTS.md 技能规范
- 添加 MIT 许可证
