## 各平台详情

### 1. Google Patents ⭐ 首选
- 网址：https://patents.google.com
- 方式：API 直连（patent-downloader SDK）
- 依赖：`pip install patent-downloader`
- 登录：不需要；验证码：无；IP 限制：无
- 速度：快（秒级）；覆盖：全球专利（中国专利为主要测试场景）
- 支持格式：公告号 `CN223081266U` / 申请号 `202421964517.8`（自动匹配）
- 更新时效：公开后 1 天内收录
- 批量：支持（传入列表或文件）

### 2. 度衍专利 (uyanip)
- 网址：https://www.uyanip.com
- 方式：浏览器自动化（Playwright）
- 登录：需要（环境变量，见 `config/.env.example`）
- 验证码：无（搜索引擎结果页无验证码）
- 特点：搜索结果直接有 PDF 下载按钮
- 劣势：需浏览器自动化，比 API 慢

### 3. 润桐 RainPat（等待恢复）
- 网址：https://www.rainpat.com
- 状态：⏸ 2026-07-11 时服务器硬件升级中，预计 2026-07-17 恢复
- 特点：永久免费、无下载限制
- API：有开放 API（需"桐钱"）

### 4. 专利公布公告系统 (epub)
- 网址：http://epub.cnipa.gov.cn
- 方式：浏览器自动化
- 登录：不需要
- 验证码：有（需处理验证码/反爬）

### 5. PatentStar（失效）
- 网址：https://cprs.patentstar.com.cn
- 登录：可以登录，但搜索 API 返回 Ret=206 内部错误
- 状态：❌ 搜索接口已失效，所有查询格式均不可用
- 注意：商用系统，接口可能经常变动

### 6. 粤港澳/PSS（待完善）
- PDF 下载环节尚未完成
- 账号通过环境变量配置，见 `config/.env.example`
