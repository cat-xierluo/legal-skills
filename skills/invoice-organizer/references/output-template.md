# 清单输出模板

发票信息整理后的清单输出规范。**默认出具报销清单**，也可切换为消费清单、对账流水等其他形态（改标题与头部表即可）。本文件以报销清单为默认示例。

## 清单文件命名

`YYMMDD 报销清单.md`

- YYMMDD = 预计发送日期（询问用户；无指定则取当天）
- 其他形态对应改名，如 `YYMMDD 消费清单.md`

## 清单结构（报销清单·默认）

````markdown
# 报销清单 — YYYY-MM 事由简述

## 报销信息

| 项目 | 内容 |
|------|------|
| **事由** | ... |
| **报销单位（发票抬头）** | 公司名称（税号） |
| **期间** | YYYY-MM-DD ～ YYYY-MM-DD |
| **案号** | ...（关联案件时填） |
| **出差人** | 姓名（差旅场景填，只写姓名） |
| **路线** | A ⇌ B（差旅场景填） |

## 凭证明细

| 序号 | 日期 | 类型 | 摘要 | 金额(元) | 发票号 |
|------|------|------|------|----------|--------|
| 1 | 2026-06-15 | 交通-高铁 | G1220 苏州北→石家庄 二等座 | 548.00 | ... |
| 2 | 2026-06-16 | 餐饮 | XX餐厅 部门聚餐 | 320.00 | ... |
| 3 | 2026-06-17 | 住宿 | 石家庄XX酒店 2 晚（价税合计） | 516.21 | ... |
| | | | **合计** | **1,384.21** | |
````

头部"报销信息"表字段按场景灵活：事由/报销单位/期间常用；案号、出差人、路线为差旅等特定场景的可选字段，无关则省略，不强填"无"。

## 其他清单形态

切换清单类型时改标题与头部表，凭证明细表结构不变：

- **消费清单**：标题 `# 消费清单 — ...`，头部可只留"消费方/期间"，明细同
- **对账流水**：标题 `# 对账流水 — ...`，按时间排序，可加"交易对象/收支方向"列
- **按月汇总**：按类型或项目聚合金额，出统计表

## 风格规则

- 不加 emoji
- 不加"住宿费说明""备注"等说明性文字段落
- 凭证明细列固定：序号 / 日期 / 类型 / 摘要 / 金额(元) / 发票号
- 涉及人员只写姓名，不带身份证号等证件信息
- 类型枚举：交通-高铁 / 交通-航空 / 交通-网约车 / 住宿 / 餐饮 / 办公采购 / 其他
- 金额保留两位小数，合计行加粗

## 发票归档命名

`YYMMDD 类型 摘要 [人名].pdf`

- YYMMDD = 发票业务日期；住宿用入住-退房区间（`YYMMDD-YYMMDD`）
- 示例：
  - `260615 高铁票 苏州北-石家庄 G1220 张某.pdf`
  - `260615-0617 住宿费 石家庄XX酒店.pdf`
  - `260620 餐饮费 XX餐厅 部门聚餐.pdf`

## 归档目录与操作

- 默认归档目录：项目内 `报销/`；差旅等可进一步子分类（如 `报销/差旅/`）
- 复制（cp）而非移动，保留原件；原件是否清理由用户决定，不擅自删除
