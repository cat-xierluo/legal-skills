# SVG Book Illustrator — 并入 svg-article-illustrator

> 本文件记录 svg-book-illustrator 的核心工作流，已并入 `svg-article-illustrator`。

## 核心定位

为书籍章节和正式文章生成简洁专业的 SVG 技术配图。风格为白底简洁专业风，适合纸质出版。

## 布局模板（8 种）

| 模板 | 适用场景 | 典型元素数 |
|------|---------|-----------|
| **flow** | 流程图、步骤图、管道图 | 3-5 个节点（水平≤4） |
| **layer** | 层次架构、分层堆叠 | 3-4 层 |
| **matrix** | 前后对比、并排比较 | 2 列 |
| **hub** | 中心辐射、生态关系 | 1 核心 + 4-8 外围 |
| **tree** | 层级结构、组织图、金字塔 | 3 层 |
| **cycle** | 循环流程、迭代闭环 | 4-6 个节点 |
| **flow+matrix** | 递进流程附带阶段对比 | 3-4 阶段 + 对比区 |
| **flow+hub** | 编排流程中节点展开 | 主流程 + 展开节点 |

## 设计规范

- **画布**：720×400，白底，安全边距 40px
- **风格**：简洁、专业、静态（无动画、无渐变、无滤镜、无 emoji）
- **颜色**：深灰线条 + 单一强调色（每图一个）
- **文字**：PingFang SC / Microsoft YaHei，节点标签 18px 起
- **形状**：圆角矩形（rx="6"）、简洁箭头、最小 24px 间距
- **印刷**：黑白可辨，颜色不是唯一区分手段

## 工作流程

1. 读取章节 Markdown 文件
2. 匹配 `references/diagram-catalog.md` 中的预定义插图
3. 识别适合配图的位置（架构/流程/对比/层次/循环/生态描述处）
4. 插入占位符 `[[FIG:N:简要描述]]`
5. 选择布局模板，按规范生成 SVG
6. 替换占位符，添加图注 `**图 N-X：图标题**`
7. 提取 SVG 到独立文件归档

## PNG 导出

```bash
node scripts/svg2png.js input.svg [output.png] [dpi]
# 默认 600 DPI，支持 emoji、中文、CSS
```
