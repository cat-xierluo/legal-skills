# 课程主题体系（精简索引）

> 来源：杨卫薪律师法律AI课程主题体系.md
> 更新日期：2026-04-23

## 阶段一：认知基础（2 讲，约 3 小时）

- 第 1 讲：法律 AI 全景图 — Chatbot→Agent 演进、AI 产业分层架构、产品地图、场景识别、上下文工程
- 第 2 讲：Agent 基础认知 — Chatbot→Agent 范式转移、平台生态、核心能力

## 阶段二：基础工具与技能（4 讲，约 3-4 小时）

- 第 3 讲：AI 时代三大基建 — Markdown / 语音 / OCR
- 第 4 讲：提示词工程基础 — 核心原则、法律场景示例、避坑
- 第 5 讲：法律 AI 知识库搭建与运用 — 搭建方法论、ima 实操、联动
- 第 6 讲：本地客户端与 Chatbot 使用实战 — 客户端选型、API 配置、Chatbot 搭建

## 阶段三：Agent 进阶与 Skill 开发方法论（4 讲，约 5-6 小时）

- 第 7 讲：Agent 进阶实践 — OpenClaw/Claude Code 安装部署、能力演示
- 第 8 讲：Skill 的本质与设计思维 — 定义、分层架构、设计思维
- 第 9 讲：Skill 开发六步法（上）— 识别、规划、编写
- 第 10 讲：Skill 开发六步法（下）— 验证、迭代、编排

## 阶段四：实战案例（2 讲，约 2-3 小时）

- 第 11 讲：法律 Skill 实战精选 — 合同审查、判决分析、服务方案、证据目录
- 第 12 讲：知识资产化与效率工具 — 资产化路径、模型选型、工作机配置

## 阶段五：组织落地（2 讲，约 2-3 小时）

- 第 13 讲：企业视角与行业变革 — 交付能力、95% 案件论、窗口期
- 第 14 讲：安全合规与规模化部署 — 数据安全、合规框架、部署经验

## 阶段六：超级个体工作流（4 讲，约 5-6 小时）

- 第 15 讲：Agent 工作空间搭建 — MacBook 服务器、工作区、模型策略
- 第 16 讲：移动办公与任务编排 — 坚果云同步、Discord 编排、定时自动化
- 第 17 讲：Git 版本控制与知识管理 — Git 项目管理、Obsidian、分支模型
- 第 18 讲：Skill 深度实践与商业化 — Command→Skill 演进、上下文优化、维保模型
