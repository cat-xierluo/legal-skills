# 总览文档生成提示词

基于课程大纲，生成课程总览文档，帮助学习者快速了解课程全貌。

## 核心要求

1. **段落式写作**：完全避免要点列表，使用段落形式
2. **全面覆盖**：涵盖课程概述、学习目标、课程结构、学习路径
3. **段落式展开**：每个部分都以段落形式详细阐述
4. **客观陈述**：禁止使用"主讲人强调"、"主讲人指出"、"主讲人提到"等表述，直接呈现课程内容本身

## 输出格式

```
# [课程标题] - 总览

## 一、课程概述
[基于内容分析的简要概述，以段落形式展开介绍课程背景、核心价值和主要内容简介。避免使用要点列表，用段落形式完整阐述课程的背景、意义和价值。]

## 二、学习目标
本课程旨在帮助学习者达成以下目标：
首先，掌握核心概念和要点，建立系统化的知识框架；
其次，理解各主题之间的内在关联，形成完整的知识体系；
最后，通过实践应用，将理论知识转化为实际工作能力。

## 三、课程结构
本课程共分为X个章节，每个章节聚焦一个核心主题。课程采用循序渐进的结构设计，从基础概念到实践应用，帮助学习者逐步建立完整的知识体系。

第一章主要探讨[主题A]，为后续学习奠定基础。第二章深入分析[主题B]，揭示其核心原理和实践价值。第三章着重讲解[主题C]，通过案例演示其实际应用。各章节之间既有独立性，又相互关联，共同构成完整的知识图谱。

## 四、学习路径建议
为确保学习效果，建议按照以下路径进行学习：
首先，从基础概念入手，理解相关理论背景和技术原理；
其次，通过案例分析，加深对知识点的理解和应用能力；
最后，结合实际工作场景，将所学知识转化为实践技能。
```

## 输入内容

- 课程大纲：[生成的课程大纲]
- 课程标题：[AI 生成的课程名称]
