# 布局示例

## per-page=2（A4 横版，左右并排）

适用于手机截图（竖版 1206×2622 等）。

```
┌─────────────────────────────────────────┐
│           A4 横版 842×595 pt             │
│  ┌──────────┐   margin   ┌──────────┐   │
│  │          │             │          │   │
│  │  图 1    │             │  图 2    │   │
│  │          │             │          │   │
│  │          │             │          │   │
│  └──────────┘             └──────────┘   │
│                 margin                    │
└─────────────────────────────────────────┘
```

## per-page=3（A4 横版，三列并排）

适用于视频截图（竖版手机录制画面）。

```
┌───────────────────────────────────────────────┐
│              A4 横版 842×595 pt                │
│  ┌────┐   ┌────┐   ┌────┐                     │
│  │    │   │    │   │    │                     │
│  │ 1  │   │ 2  │   │ 3  │                     │
│  │    │   │    │   │    │                     │
│  │    │   │    │   │    │                     │
│  └────┘   └────┘   └────┘                     │
└───────────────────────────────────────────────┘
```

## per-page=1（A4 单张，自动横竖）

根据图片方向自动选择 A4 横版或竖版。

```
竖版图片:              横版图片:
┌──────────┐          ┌─────────────────────┐
│          │          │                     │
│          │          │                     │
│   图片   │          │       图片          │
│          │          │                     │
│          │          └─────────────────────┘
│          │
└──────────┘
A4 竖版              A4 横版
```

## margin 效果

`--margin` 控制图片与页面边缘的距离。默认 25pt（约 8.8mm）。

- `--margin 0`：图片紧贴页面边缘（不推荐打印）。
- `--margin 25`：标准页边距（默认）。
- `--margin 40`：较宽页边距，适合装订。
