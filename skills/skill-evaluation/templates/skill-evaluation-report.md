## 六、总体结论

- [ ] ✅ 进入工作流：结构可评测、内容评测通过、taste 项无致命命中
- [ ] ⚠️ 修后再测：存在低分维度或 taste 命中，按"要补的句子"清单修补后重测

### 回退指引

- Hard Fail 命中 → 回 skill-lint 修 Hard Fail。
- 场景识别不清（Skill 声称覆盖过宽） → 回 ch07 第三节"识别场景"，先把场景收窄。
- 结构性缺口过大（五项可评估性基础缺三项以上） → 回 skill-lint 补可评估性基础。
