# 企业聚合总览

> POST /open/rh_enterpriseAggregationSummary
> 计费: 10 积分/次

获取企业各维度数据汇总，一次调用返回多项统计摘要。

## 请求参数（Body）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | string | 否 | 企业 ID |
| uscc | string | 否 | 统一社会信用代码 |

### 校验规则

- id 和 uscc 至少提供一个
- 若同时提供，以 id 为准

## 返回结构

返回 `data` 对象，包含企业各维度信息的聚合统计摘要（涉诉数量、投资数量、商标数量等）。
