# 企业基本信息

> GET /open/rh_enterpriseBaseInfo
> 计费: 10 积分/次

根据企业 ID 或统一社会信用代码获取企业完整基本信息，包含股东、核心成员、分支机构等。

## 请求参数（Query）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | string | 否 | 企业 ID |
| uscc | string | 否 | 统一社会信用代码 |

### 校验规则

- id 和 uscc 至少提供一个
- 若同时提供，以 id 为准

## 返回结构

返回 `data` 对象，包含以下扁平化信息：

- 基本信息（企业名称、信用代码、类型、状态、法定代表人等）
- 股东信息
- 十大股东
- 十大流通股东
- 核心成员
- 分支机构

返回为嵌套 JSON 对象，各子模块字段名称以中文为主。
