# 企业欠税公告记录列表

> GET /open/rh_enterpriseCorporateTax
> 计费: 5 积分/次

查询企业欠税公告记录，支持分页。

## 请求参数（Query）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | string | 否 | 企业 ID |
| uscc | string | 否 | 统一社会信用代码 |
| page | int | 否 | 页码（默认 1） |
| size | int | 否 | 每页条数（默认 30） |

### 校验规则

- id 和 uscc 至少提供一个
- 若同时提供，以 id 为准

## 返回结构

返回分页列表，包含欠税公告记录（欠税税种、欠税余额、发布单位、发布日期等）。
