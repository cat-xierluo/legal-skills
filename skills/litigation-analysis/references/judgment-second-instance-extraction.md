# 二审判决书处理文档

本文档定义了二审判决书的完整处理流程，包括识别特征、抽取规范、处理流程和输出侧重。

---

## 识别特征

### 二审判决书特征

- **案号格式**："(20XX)X民终XXXX号"
- **章节结构**：一审认定、二审新查明、本院认为、判决如下
- **终审说明**：包含"本判决为终审判决"
- **篇幅**：通常 100-300 行

---

## 抽取规范

二审判决书是 litigation-analysis 的核心处理对象之一，采用结构化深度抽取。

### 抽取原则

- **对比分析**：需要对比一审判决与二审判决的差异
- **聚焦变化**：重点识别二审新证据、新查明事实、改判理由
- **终审导向**：分析对再审或执行的影响

抽取时需要：完整提取、对比分析、结构化呈现

### 基础信息抽取

| 字段 | 提取规则 | 说明 |
|------|---------|------|
| 案号 | 正则匹配"(20XX)X民终XXXX号" | 二审用"终" |
| 法院 | 从法院名称中提取完整名称 | 包含省市区信息 |
| 案由 | 从文书标题或内容中提取 | 识别案件性质 |
| 上诉人 | 提取上诉人名称（原审原告/被告） | 明确上诉地位 |
| 被上诉人 | 提取被上诉人名称（原审原告/被告） | 明确应诉地位 |
| 审理程序 | 二审终审 | 明确终审性质 |
| 裁判日期 | 从文书末尾提取 | 判决生效日期 |

### 事实认定抽取

**一审认定**：
- 从上诉状和被上诉人答辩中提取一审判决的核心认定
- 一审法院查明的事实
- 一审争议焦点

**二审新查明**：
- 二审新证据（新提交的证据）
- 二审新查明的事实（与一审不同的认定）
- 与一审判决的差异点

**二审争议焦点**：
- 上诉请求（上诉人的上诉理由）
- 被上诉人答辩
- 二审法院归纳的争议焦点

### 裁判理由抽取

**本院认为**：
- 对比一审判决的变化
- 分析改判/维持的理由
- 法律适用和说理逻辑

**法律依据提取**：
- 法律名称（引用的法律法规名称）
- 条款号（具体条款号）
- 条文内容（关键条文内容）
- 司法解释（引用的司法解释）

### 裁判结果抽取

**判决主文**：
- 维持一审判决第X项
- 撤销一审判决第X项
- 变更一审判决第X项为：XXX
- 新增判决项

**上诉权利告知**：
- 本判决为终审判决

## 处理流程

### 二审判决处理流程

1. **基础信息提取**：提取案号（"终"字标识）、法院、当事人等
2. **一审认定提取**：从上诉状和被上诉人答辩中提取一审判决的核心认定
3. **二审新查明提取**：识别二审新证据和新查明事实
4. **对比分析**：对比二审判决与一审判决的差异
5. **裁判理由提取**：提取改判/维持的理由和法律依据
6. **裁判结果提取**：提取判决主文、终审说明
7. **质量检查**：完整性、准确性、逻辑性检查

## 输出侧重

### 内部版

- 📋 核心要点速览（含与一审判决对比）
- 一审与二审认定差异
- 争议焦点与法院认定（逐一对应）
- 判决结果详解
- 法律分析与再审策略建议
- 裁判要点与实务启示

### 研究版

- 识别的法律问题
- 再审理由研究
- 改判规则总结
- 类似案例检索方向

### 客户版

- 二审结果通俗解释
- 与一审判决的差异说明
- 执行建议
- 风险提示
- 建议行动

---

## 抽取质量检查

### 完整性检查

- [ ] 基础信息：案号、法院、当事人、日期齐全
- [ ] 一审认定：一审判决核心认定完整
- [ ] 二审变化：新证据、新查明事实已提取
- [ ] 裁判理由：改判/维持理由清晰
- [ ] 裁判结果：判决主文逐项列出，无遗漏

### 准确性检查

- [ ] 案号格式正确（"(20XX)X民终XXXX号"）
- [ ] 法院名称完整（包含省市区）
- [ ] 当事人地位准确（上诉人/被上诉人不混淆）
- [ ] 法条引用完整（法律名称+条款号）
- [ ] 判决项与原文一致（无遗漏或错位）

### 逻辑性检查

- [ ] 二审认定与一审认定的差异清晰
- [ ] 改判/维持理由与争议焦点匹配
- [ ] 判决结果与裁判理由一致
- [ ] 终审效力明确
