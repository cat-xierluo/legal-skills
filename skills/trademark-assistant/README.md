# trademark-assistant

面向中国商标申请的类别规划、可注册性初筛及申请材料准备，基于尼斯分类和中国商标法律法规。

> 从"我该注册哪些类别"到输出结构化初筛报告和商标申请材料，一步到位。

## 适合谁用

- 需要快速判断商标可注册性的企业和个人
- 进行商标类别规划的律师和代理人
- 准备商标申请材料（商品清单、商标说明）的从业者

## 典型场景

```text
用户：我想注册一个叫"星火"的商标，做教育软件的，帮我看看能不能注册，需要注册哪些类别
AI：我来帮你做尼斯分类类别规划、可注册性初筛，并输出结构化风险分级报告
```

## 它能产出什么

- 按尼斯分类的商标类别规划表
- 可注册性初筛报告（含风险分级）
- 商标说明、商品清单等申请材料

## 安装方式

1. 打开本仓库的 GitHub Releases。
2. 下载最新版本的 skill 压缩包。
3. 解压后将 `trademark-assistant/` 文件夹放入你的 skill 目录。
4. 在支持 `SKILL.md` 的 Agent / Claude 环境中启用该 skill。

## 使用边界

**适合：**
- 中国商标申请的类别规划
- 可注册性初步筛查
- 商标申请材料准备

**不适合：**
- 替代律师/代理机构的正式法律意见
- 承诺注册成功率
- 处理复杂商标争议（抢注、驰名商标等）
