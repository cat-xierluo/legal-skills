# 材料分类详细指南

本文档补充 `assets/*.yaml` 中 `classification` 关键词匹配之外的分类决策逻辑。

## 分类原则

1. **顺序匹配**：按 YAML 中 `classification` 列表顺序，命中第一条即停止
2. **文件名优先**：匹配对象为文件名（含扩展名），不区分大小写
3. **内容辅助**：文件名无法判断时，可读取文件内容前几行辅助判断

## 模糊场景处理

### 录音归属：律师-客户 vs 客户-对方

- **律师与客户的沟通录音** → 01-客户材料（属于客户提供的咨询记录）
- **客户与对方的交涉录音** → 01-客户材料（客户提供的事实材料）
- 录音来源无法判断时，默认归入 01-客户材料

### 法律服务方案

法律服务方案统一放在 **02-工作文件**，不放在根目录。方案属于律师产出，且经常迭代更新。

### 视频文件归类

- 客户与对方沟通的现场录像 → 01-客户材料
- 客户拍摄的证据固定视频 → 01-客户材料
- 律师与客户的咨询录像 → 01-客户材料

### 转化后的子案件文件夹

咨询项目转化为诉讼案件后，可能产生以法院案号命名的子文件夹。处理方式：
- 保留在咨询项目根目录下，不归类到 01/02
- 在项目信息文件的"关联案件"部分记录子案件编号和路径
- 子文件夹内部结构遵循 litigation 预设

### 录音与转写文稿配对

转写文稿与对应录音必须放在**同一目录**。如果录音在 01-客户材料，转写也放 01。

### 无法判断的文件

归入最相关的目录：
- 咨询项目 → `01 - 📂 客户材料`（默认归入客户材料）

## 商标项目分类（项目共享 + 申请号子目录）

### 项目共享 vs 申请号子目录归属原则

商标项目有**两层目录**：

- `00 - 🤝 项目共享/`：跨申请号共享的文件，一次性提供、整案复用
- `{申请号}/` 下的 7 个子目录：申请号专属文件

判断文件进哪一层：

| 文件类型 | 归属 | 理由 |
|----------|------|------|
| 商标代理委托书（首签） | 00 项目共享 | 一次签、所有申请号共用 |
| 商标代理合同 | 00 项目共享 | 整案签 |
| 营业执照、身份证、律所执业证 | 00 项目共享 | 主体资格文件 |
| 共享图样（多申请号共用同一标识） | 00 项目共享 | 跨申请号共用 |
| 商标评审代理委托书（复审专用） | 申请号 → A-01 委托材料 | 该复审专属 |
| 商标异议代理委托书（异议专用） | 申请号 → A-01 委托材料 | 该异议专属 |
| 申请号专属图样 | 申请号 → A-02 商标图样 | 单申请号 |
| 申请书（注册/异议/复审/无效） | 申请号 → A-03 申请书 | 单申请号 |
| 商标局文书 | 申请号 → A-04 官方文书 | 按号定位 |
| 异议/无效证据 | 申请号 → A-05 证据材料 | 单申请号 |
| 检索/分析/方案 | 申请号 → A-06 律师工作文件 | 通常按申请号 |
| 律师费发票 | 申请号 → A-07 发票/财务 | 财务独立记账 |

### 业务子类型与目录选择

5 种业务子类型不创建独立预设，但分类规则略有差异：

| 业务类型 | 必含目录 | 通常不含 |
|----------|----------|----------|
| 商标注册 | A-02、A-03、A-04（受理/补正/缴费/驳回/注册证） | A-05 证据 |
| 商标异议 | A-02、A-03、A-05（异议证据）、A-04（决定书） | - |
| 商标复审 | A-03（复审申请书）、A-04（驳回通知、复审决定） | - |
| 商标无效 | A-03（无效请求/答辩）、A-05（无效证据）、A-04（裁定） | - |
| 商标变更 | A-03（变更/转让申请书）、A-04（变更证明） | A-05 证据 |

### 模糊场景处理

#### 商标图样（jpg/png）归属

- 多申请号共用同一图样 → `00 项目共享/共享图样/`
- 单申请号专属图样 → `{申请号}/02 - 🖼️ 商标图样/`
- 无法判断 → 默认归入 `{主申请号}/02 - 🖼️ 商标图样/`

#### "商标驳回通知书" vs "驳回复审申请书"

两者**不**放在同一目录：
- 驳回通知书（商标局发的） → `{申请号}/04 - 🏛️ 官方文书/`
- 驳回复审申请书（律师起草的） → `{申请号}/03 - ✍️ 申请书/`

#### 商标代理委托书版本

委托书有多版本（.doc / .jpg / .pdf / 盖章版）—— 全部放 `00 项目共享/`，**不要**分散到各申请号子目录。

#### 引证商标材料

异议/复审/无效案件中引证他人商标的材料，分类逻辑：
- 引证商标号、申请人、注册证复印件 → `{申请号}/A-05 证据材料/`（作为证据一部分）
- 引证商标的查询分析 → `{申请号}/A-06 律师工作文件/`

#### 商标监测报告

定期对客户商标做状态监测的报告（如潽洛因思的"商标注册状态.docx"）：
- 项目级监测 → 视情况进 `00 项目共享/` 或 `{主申请号}/A-06`
- 申请号专属 → `{申请号}/A-06 律师工作文件/`

## 诉讼 vs 咨询 vs 商标分类差异

| 场景 | 诉讼归类 | 咨询归类 | 商标归类 |
|------|----------|----------|----------|
| 起诉状 | 06-法律文书 或 08-法院送达 | 01-客户材料 | 不适用（商标无起诉状） |
| 委托合同 | 01-委托材料 | 01-客户材料 | 00 项目共享 |
| 法律服务方案 | 01-委托材料 或根目录 | 02-工作文件 | 视情况进 A-06 律师工作文件 |
| 咨询录音 | 一般不出现 | 01-客户材料 | 一般不出现 |
| AI 咨询文档 | 一般不出现 | 01-客户材料 | A-06 律师工作文件（AI材料） |
| 客户-对方交涉录音 | 一般不出现 | 01-客户材料 | A-05 证据材料（如有） |
| 商标代理委托书 | 不适用 | 不适用 | 00 项目共享 |
| 商标驳回通知书 | 不适用 | 不适用 | {申请号}/A-04 官方文书 |
| 商标驳回复审申请书 | 不适用 | 不适用 | {申请号}/A-03 申请书 |
| 商标图样 | 不适用 | 不适用 | 00 项目共享 或 {申请号}/A-02 |
