# Changelog

## [1.1.0] - 2026-02-15

### 新增
- **去重检查机制**：每个 Subagent 检索前需声明关键词，主 Agent 审核后确认无重复再开始
- 动态调整：当发现方向重复时，通知 Agent 转向其他角度
- 重命名为通用技能名称 multi-search（多主题深度研究）

### 改进
- 优化上下文传递，增加已分配关键词方向信息

## [1.0.0] - 2026-02-15

### 新增
- 从 Command 迁移为 Skill，保留完整功能
- 采用智能输出目录检测机制（优先 output/，其次当前目录，兜底 ./research/）
- 保持并行 Subagent 架构，支持完全自动化执行
