# 商标案件预设（trademark）设计方案

**日期**：2026-06-08
**作者**：brainstorm session
**目标**：在 new-case skill 中新增 `trademark` 预设，覆盖商标注册/异议/驳回复审/无效宣告/变更转让业务

## 背景

`new-case` skill 现有 `litigation`（12 目录）和 `consultation`（2 目录）两个预设，文档明确预留了"知产申请"扩展点。律师 `~/Documents/Mac同步文件夹/工作文档/006 - 商标/` 现有 20+ 个商标项目，目录结构散乱，新项目全部平铺，老项目模式不统一，无法直接套用现有预设。

## 决策摘要

### DEC-T01：单一预设 + 业务类型字段

5 种子业务（注册 / 异议 / 驳回复审 / 无效宣告 / 变更转让）通过元数据字段表达，不拆子预设。
- 理由：复杂项目（潽洛因思型）同时含多种业务，拆预设会导致目录分裂；目录结构差异主要在材料内容而非组织逻辑。

### DEC-T02：7 个申请号子目录平铺

项目根目录下，每个商标申请号一个子目录；申请号子目录内是 7 个固定目录平铺，不再下钻。

### DEC-T03：项目共享区与申请号专属分离

委托书、合同、营业执照、律所执业证等"一次签、整案用"的文件统一进 `00 - 项目共享/`；申请书、官方文书、证据材料等申请号专属文件进申请号子目录。

### DEC-T04：项目命名 `{YYMMDD} 客户简称 业务类型`

- 例：`251119 方帅体育 商标注册`
- 多商标/多申请号的项目不把商标名放进项目名；商标名进项目信息卡

### DEC-T05：老项目不迁移

新预设只用于新建项目。2024 年及之前有手工整理目录的老项目保留现状。

## 目录结构

```
{YYMMDD} {客户简称} {业务类型}/
├── [{YYMMDD}] 项目信息.md         ← 项目信息卡（trademark-info 模板）
├── [{YYMMDD}] 待办事项.md         ← 待办清单
├── 00 - 项目共享/                  ← 跨申请号共享
│   ├── 委托书（商标代理委托书 / 商标评审代理委托书 / 商标异议代理委托书）
│   ├── 合同（商标代理合同）
│   ├── 主体资格（营业执照、身份证、律所执业许可证、公证委托、商标许可授权书）
│   └── 共享图样（多申请号共用同一标识时）
├── {申请号-1}/                    ← 每个申请号一个子目录
│   ├── 01 - 委托材料/             ← 申请号专属委托（评审/异议委托书等）
│   ├── 02 - 商标图样/             ← 该申请号图样（黑白/彩色多版本）
│   ├── 03 - 申请书/                ← 律师起草（注册/异议/复审/无效/暂缓审查申请书、证据目录、答辩材料目录）
│   ├── 04 - 官方文书/              ← 商标局文书（受理/补正/缴费/驳回/公告/注册证/决定书/裁定书/变更证明）
│   ├── 05 - 证据材料/              ← 异议/无效/复审案件证据
│   ├── 06 - 律师工作文件/          ← 检索/分析/状态/方案/AI材料
│   └── 07 - 发票/财务/             ← 律师费发票、官费发票
└── {申请号-2}/...
```

## 商标信息卡（trademark-info.md）字段

### 顶部：基本信息

| 字段 | 必填 | 说明 |
|------|------|------|
| 客户/申请人 | 必填 | 公司全称 |
| 商标名称 | 必填 | 文字商标或图形商标命名 |

## 分类规则（classification）

参考 consultation 预设的写法，按文件名关键词匹配：
- 委托书、合同、营业执照、身份证 → `00 - 项目共享/`
- 商标图样（jpg/png） → `02 - 商标图样/`
- 申请书、复审申请书、异议申请书、无效宣告请求/答辩书、暂缓审查申请书、证据目录、答辩材料目录 → `03 - 申请书/`
- 商标局文书（受理/补正/缴费/驳回/公告/注册证/决定书/裁定书/变更证明） → `04 - 官方文书/`
- 微信截图、展会照片、交易票据、抢注信息表、证据材料、证据 N → `05 - 证据材料/`
- 检索报告、查询报告、注册状态、驳回分析、审查意见、申请规划、报价方案、AI材料 → `06 - 律师工作文件/`
- 律师费发票、官费发票、缴费通知 → `07 - 发票/财务/`
